# Cold-Email-Generator
An end-to-end Generative AI project that helps Software and AI services companies create personalized cold emails for potential clients.
This tool leverages open-source LLMs, retrieval-augmented generation (RAG), and an intuitive Streamlit UI to automate and scale high-quality outreach.

Features
Llama 3.1 open-source LLM for high-quality email generation
ChromaDB vector store for storing and retrieving client/company data
LangChain for orchestrating prompts, pipelines, and RAG
Streamlit frontend for a clean, interactive user experience
End-to-end workflow from data ingestion → vector embedding → RAG → email generation
Customizable email tone, style, and target audience
Fast, reproducible, modular architecture
